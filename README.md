# Attendance Management System

A Python-based Attendance Management System that stores student and attendance records using CSV files.

## Features

* Add Student
* Remove Student
* Mark Attendance
* View Students
* View Student Attendance
* View Attendance by Date
* Calculate Attendance Percentage
* Prevent Duplicate Student Entries
* Prevent Duplicate Attendance on the Same Day
* Text-to-Speech Notifications

## Technologies Used

* Python
* CSV Module
* OS Module
* Time Module
* Win32com Client (Text-to-Speech)

## Project Structure

Attendance-Management-System/

├── main.py

├── students.csv

├── attendance.csv

└── README.md

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:

python main.py
